SimpleHeal

Este plugin permite curarte a ti mismo o a otros jugadores mediante el comando /heal o /heal <player>. Es fácil de usar y perfecto para servidores donde el control de la salud es importante.

Ademas puedes configurar el precio de usarlo en caso de usar Vault y Essentials. Tambien puedes cambiar los segundos que deben pasar para que un jugador pueda volver a usar el comando /heal.

Puedes activar y desactivar la opcion de pagar y la opcion de esperar antes de volver a usar el comando /heal.

Todos los mensajes se pueden modificar.
